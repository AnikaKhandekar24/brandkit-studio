# BrandKit Studio

Build your personal brand in minutes.

BrandKit Studio is Project 01 in CreatorStack by Anika, a series of useful tools for students, creators, freelancers, small businesses, and personal brands.

**Live site:** [anikakhandekar24.github.io/brandkit-studio](https://anikakhandekar24.github.io/brandkit-studio/)

## Features

- Rule-based personal brand generation with no API
- Bio, tagline, portfolio intro, LinkedIn About, social bio, and project copy
- Six curated color palette directions
- Live personal brand preview
- Local saved kits
- One-click copy and text download
- Responsive creator-focused interface

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Built with React and Vite for CreatorStack by Anika.
